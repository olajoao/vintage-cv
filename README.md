# joaokuster.vercel.app

Personal portfolio — retro-vintage single-page site.

## Stack

- Static HTML/CSS/JS (zero dependencies, no build step)
- Deployed on Vercel

## Design

Retro-vintage editorial aesthetic with a warm palette (`#E06A4E`, `#DEB853`, `#E2D9A2`, `#789F8A`, `#0A373A`). Typography: Fraunces (display), Syne (headings), Newsreader (body) via Google Fonts. Grain texture overlay, vintage decorative frames, scroll-triggered animations, and `prefers-reduced-motion` support.

## Sections

Hero · About · Skills · Experience · Projects · Education · Contact

## Deploy

```bash
vercel
```

Or connect the repo to Vercel — it serves `index.html` directly, no build required.

## Local

Open `index.html` in a browser, or:

```bash
npx serve .
```
